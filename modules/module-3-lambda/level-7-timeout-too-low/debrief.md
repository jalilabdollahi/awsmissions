# Three Seconds and Out

## Why This Matters
This mission practices a real troubleshooting pattern in Lambda Functions: Function needs 30 seconds to process but timeout is 3s — always times out

## Production Lesson
In real AWS incidents, the fastest path is usually to compare actual state with expected state, then fix the smallest misconfiguration that restores service safely.
